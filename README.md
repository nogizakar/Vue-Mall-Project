## git常用命令

### 查看提交历史

```
git log
```

### 查看状态

```
git status
```

### 已初始化

```
git init
```

### 添加所有新修改文件

```
git add .
```

### 提交到本地仓

```
git commit -m 'xxx'
```

### 将本地仓与github远程仓关联

```
git remote add origin git@github.com:chenxuange/mall-learn.git
```

### 解除本地仓与远程仓的关联

```
git remote rm origin
```

### 将本地仓与gitee远程仓关联

```
git remote add origin git@gitee.com:valerius/mall-learn.git
```

### 重命名当前分支

```
git branch -M main
```

### 推送到远程main（远程同步创建）

```
git push -u origin main
```

### 拉取

```
git pull
```
